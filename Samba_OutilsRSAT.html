Installation des outils de gestion AD à distance (RSAT)
1 - Installation.
2 - Lancement et configuration
1 - Installation :

Pour les versions de Windows 10 avant 1809 : il faut télécharger RSAT sur le site de Microsoft

Après la version 1809, RSAT est intégré à Windows 10, il suffit de l'activer.

Paramètres Windows / Applications.

Fonctionnalités facultatives

Cliquer sur "Ajouter une fonctionnalité"

sélectionner les fonctionnalités RSAT que vous voulez installer

Comme:

RSAT: outils de gestion de stratégie de groupe

2 - Lancement & Configuration :

Il existe plusieurs façon de lancer l'interface, on peut par exemple lancer une console MMC

"Touche Windows" + R, taper mmc puis entrée

Ensuite menu "Fichier", "Ajouter:supprimer un composant logiciel enfichable"

Cliquer dans la liste puis sur 'Ajouter', le choix du composant comporte en plus le choix de la cible (un autre ordinateur), rentrer le nom du serveur Linux Samba 4 ou du Serveur Windows, ici 3 composants classiques :

Dossiers partagés

Utilisateur et ordinateurs Active Directory

Gestion des stratégies de groupe (GPO)

ok pour valider et revenir à la console.

On peut ensuite enregistrer la configuration sur le bureau pour l'utiliser ultérieurement(menu "Fichier", "Enregistrer sous ...").
